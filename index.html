VeloniaAngular

Gestión de productos, usuarios y pagos con Stripe

Tecnologías Principales

Angular Node.js Express MySQL Stripe JWT TailwindCSS Bootstrap

Descripción

VeloniaAngular es una aplicación web desarrollada con Angular en el frontend y Node.js + Express en el backend, utilizando MySQL como base de datos.

Permite gestionar productos, variantes, usuarios y realizar pagos seguros con Stripe. Su diseño moderno combina Bootstrap y TailwindCSS para ofrecer una experiencia limpia y responsiva.

Características

  • Gestión completa de productos (CRUD)
  • Autenticación y autorización con JWT
  • Carrito de compras funcional
  • Pasarela de pagos con Stripe
  • Interfaz responsive y diseño moderno
  • API REST propia para backend

Instalación

Nota: Antes de iniciar, importa la base de datos usando MySQL (por ejemplo, dentro de XAMPP o cualquier administrador de MySQL de tu preferencia) desde backend/database/schema.sql.
  1. Clona el repositorio: 
    git clone https://github.com/TU-USUARIO/VeloniaAngular.git
  2. Instala dependencias: 
    cd VeloniaAngular
npm install
    Para el backend: 
    cd backend
npm install
  3. Configura variables de entorno en backend/.env: 
    DB_HOST=localhost
DB_USER=root
DB_PASSWORD=tu_password
DB_NAME=velonia
JWT_SECRET=tu_jwt_secret
STRIPE_SECRET_KEY=tu_stripe_key
PORT=3000
  4. Inicia el proyecto: 
    cd backend
npm start
# En otra terminal:
ng serve

Tecnologías Utilizadas

  • Frontend: Angular, Bootstrap, TailwindCSS
  • Backend: Node.js, Express, JWT
  • Base de datos: MySQL
  • Pagos: Stripe API
